# LlamaIndex

LLM application framework providing event-driven Workflows for building agents and data connectors

## Observations

- LlamaIndex Workflows uses an event-driven architecture (rather than LangGraph's graph/Pregel model) — achieves a balance between framework structure and developer freedom, with less framework dependency; inherent async design can add complexity in some scenarios. In a direct comparison on the same multi-skill RAG/SQL/analytics agent, it was judged to offer more flexibility than LangGraph with less lock-in. — [[2025-05-27-agent-智能体开发框架选型指南]]
- By Sep 2025, community investment in LlamaIndex had "significantly contracted" per Ant Group's OpenRank analysis — alongside LangChain and AutoGen, it is cited as a primary driver of the Agent Framework domain's decline in the open source LLM ecosystem. — [[2025-09-17-从社区数据出发-再看大模型开源开发生态全景与趋势]]
