用 AI + 浏览器自动化（CDP）实战：一次抓完 857 条会议数据

我这次做了一个很纯粹的实战： 用 AI + CDM（基于 CDP 的浏览器自动化），把一个会议官网的数据完整结构化抓下来。

先简单说下 CDM： 👉 让 AI 直接进入浏览器环境，读取 DOM、分析页面、找分页、发请求，而不是手动点页面

📌 任务很简单：
打开会议页面 → 获取所有 presentation 字段 → 自动翻页 → 输出结构化数据

📌 实际过程也很干净：

1️⃣ 进入页面，确认自动化环境

2️⃣ AI 分析 DOM，识别列表结构

3️⃣ 找到分页参数（?p=），不走点击

4️⃣ 确定字段结构（title / author / session 等）

5️⃣ 批量翻页抓取

6️⃣ 最终输出 JSON

📌 结果：
总数据量：857 条
已验证抓取链路完整
数据结构非常干净

这个AI + CDM(CDP)也太牛了⬆️
