# H800

NVIDIA Hopper GPU; export-controlled variant of H100 (cut interconnect bandwidth); widely used in China.

## Synthesis

NVIDIA Hopper export-control variant of H100 — primary China-data-center GPU. Performance-relevant difference vs H100 is reduced inter-chip bandwidth; the H800 SXM5's intra-chip Tensor Core / memory characteristics are similar enough that it's the canonical Chinese-team reference platform: ByteDance's **Flux** auto-tunes across A100 + H800 / PCIe + NVLink; DeepSeek's **FlashMLA** is tuned specifically for H800 SXM5 (3.35 TB/s HBM, ~865 TFlops throttled peak).

## Observations

- One of Flux's two evaluation hardware targets (alongside A100): auto-tunable kernel-fusion comm overlap measured across A100 + H800, PCIe + NVLink combinations via CUTLASS. — [[2025-10-09-flux-fast-software-based-communication-o]]
- FlashMLA's canonical performance target — kernel achieves ~80% Tensor Core utilization and 3 TB/s memory bandwidth on H800 SXM5. The 660 TFlops headline is measured against the throttled ~865 TFlops practical peak (reduced from 990 TFlops by clock throttling to ~1600 MHz). — [[2026-01-28-flashmla-docs-20250422-new-kernel-deep-d]]
