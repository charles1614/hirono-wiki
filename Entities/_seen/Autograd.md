# Autograd

The Python autodiff library that preceded [[JAX]] as a [[Google]] Research project. Core idea — automatic differentiation of native Python/[[NumPy]] code — carried forward into JAX and extended with JIT compilation, vectorization, and multi-device parallelism.

## Synthesis

Thin (1 source). Historical ancestor, not an active target for ingests of its own. Useful context for understanding why JAX's autodiff surface feels the way it does.
