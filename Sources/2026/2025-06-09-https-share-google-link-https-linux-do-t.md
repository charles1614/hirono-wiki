# [2025-06-09] Cursor官网账号登录方式不能多人共享了？

## TL;DR

Short Linux.do discussion (April 2025) noting that [[Cursor]] switched from session-based login to access-token login, breaking multi-user account sharing; community members share a third-party client (`curs0r`) and token-based workarounds.

## Key claims

- Starting 2025-04-06, logging into Cursor via the official website kicks out other sessions sharing the same account; using access tokens allows concurrent use.
- A community-built client `curs0r-release` (GitHub: J3n5en/curs0r-release) provides token-based login with a polished UI and access to a shared account pool at `pool.curs0r.me`.
- A 30-day token alternative is also available through the official interface.
