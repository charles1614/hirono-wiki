# [2025-05-28] 新老 DeepSeek-R1、V3-0324、Qwen3-235B、豆包天气卡片代码生成对比

## TL;DR

Linux.do 社区帖子，用"生成 HTML 天气卡片动画"为 prompt 对比国内外多个模型的代码生成质量，以截图和 GIF 呈现输出效果，属于非正式社区评测。

## Key claims

- 测试 prompt：生成横板 HTML 天气页面（4 张卡片：晴天/大风/暴雨/暴雪），要求华丽外观、动画、CSS 技巧（shadow/hover/blur）。
- 参与模型：新版 [[DeepSeek-R1]]（R1-0528）、老版 R1、DeepSeek V3-0324、[[Qwen3-235B]]（思考/非思考）、豆包、Claude 3.7 Sonnet、Claude 4 Opus/Sonnet、Gemini 2.5 Pro/Flash、GPT-4.1。
- 发帖者主观评价新 R1 和豆包效果最佳；Qwen3-235B 输出被多人评为"水"/"CSS 写错了"风格；Claude 4 Opus 被评为动画效果不如 Claude 4 Sonnet；Claude 3.5 Haiku 级别也可完成类似动画。
- 讨论揭示同一模型同一 prompt 的随机性：多人指出不同 session 输出差异显著，"抽卡"效应明显；二次让模型在原代码基础上修改可显著提升动画效果。
- 本次对比仅针对"视觉美观"维度，用户明确表示"图一乐"，不代表实际代码质量。
