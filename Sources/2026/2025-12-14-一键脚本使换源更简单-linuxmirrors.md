# [2025-12-14] LinuxMirrors — One-Command Linux & Docker Mirror Switcher

## TL;DR

LinuxMirrors provides shell scripts that replace system package sources and Docker registry mirrors via a single command, with interactive arrow-key selection or headless CLI flags for CI/CD. Supports 26+ Linux distributions and is specifically curated for mainland China users, aggregating domestic educational-network mirrors and overseas mirrors.

## Key claims

- Supports 26+ Linux operating systems spanning Debian/Ubuntu family, Red Hat/CentOS/Fedora/Rocky/Alma family, openSUSE, Arch, Alpine, Gentoo, NixOS, and various Chinese domestic distros (Deepin, openKylin, Anolis, openEuler, OpenCloudOS).
- Operates with a single command (zero dependencies, no install required), with interactive arrow-key guidance or full headless automation for CI/CD.
- Claims to complete source switching in as little as 10 seconds.
- Provides domestic Chinese educational-network mirrors and curated overseas mirrors in addition to commercial CDN mirrors.
- Includes both a GNU/Linux system package source script and a separate Docker installation + mirror-switching script.
- Published under MIT license; hosted with global edge-network acceleration for reliable access from mainland China.
- Version coverage: Debian 8–13, Ubuntu 14–26, RHEL 7–10, Fedora 30–43, CentOS 7–8 + Stream 8–10, Rocky/Alma 8–10, NixOS 19–25.
