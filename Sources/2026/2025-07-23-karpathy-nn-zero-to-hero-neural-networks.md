# [2025-07-23] Neural Networks: Zero to Hero (Karpathy)

## TL;DR

A video course by Andrej Karpathy that builds neural networks from scratch, starting with backpropagation basics and culminating in a GPT transformer and BPE tokenizer. All Jupyter notebooks are open-sourced alongside the YouTube lectures.

## Key claims

- 8 lectures spanning: micrograd (backprop from scratch), makemore (bigram → MLP → BatchNorm → backprop ninja → WaveNet), GPT from scratch, and BPE tokenizer.
- Each lecture pairs a YouTube video with Jupyter notebooks and exercises hosted in the `karpathy/nn-zero-to-hero` GitHub repo.
- Lecture 7 ("Let's build GPT") follows "Attention is All You Need" and explicitly connects to ChatGPT and [[Claude Code]] / Copilot (GPT-based coding tools).
- Lecture 8 explains that LLM tokenization is a fully separate stage with its own training (BPE) and that many LLM failure modes trace back to tokenization.
- Course assumes only Python basics and high-school calculus; GPU not required for core chapters.
