# [2025-05-27] 从0到∞: 大语言模型知识面试一本通

## TL;DR

A 200+ entry living Feishu wiki covering LLM full-stack interview preparation, written by a Tsinghua CS grad who collected 15+ SSP/talent-track offers from Kuaishou, Baidu, Zhipu, ByteDance, and others. Organized into four chapters: LLM fundamentals, training-side, inference-side, and evaluation technology.

## Key claims

- Four chapters: LLM Fundamentals (Transformer-centric), Training (pretraining → SFT → RL full pipeline), Inference (CoT, RAG, Infra), Evaluation (benchmarks, RM-based methods).
- Covers [[DeepSeek-R1]] GRPO training, PPO full pipeline, DPO data synthesis, and RL algorithms through 2026-03 (GVPO, GDPO, JustRL, SAPO, multi-step Agentic RL).
- Architecture deep-dives: QK Norm, RMSNorm dominance, YaRN position encoding, Muon optimizer, MoBA (Kimi sparse attention), Native Sparse Attention ([[DeepSeek-V3]]).
- Inference-side: memory analysis for 7B model training/inference, ZeRO-1/2/3 tradeoffs, LoRA VRAM breakdown, DP/MP/PP/TP/SP/CP/EP parallelism summary.
- Tracks model releases with source-code analysis: Kimi K2 (1T), GLM-4.5/4.7, MiniMax-M1, dots.llm1 (小红书), Qwen3, Seed-Coder, Seed-Thinking-v1.5.
- Updated weekly, co-authored with Tsinghua MSc team member; intended for both beginners and interview-track engineers.
