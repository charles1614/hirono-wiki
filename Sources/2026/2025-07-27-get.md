# [2025-07-27] Microsoft Activation Scripts — get.activated.win Launcher

## TL;DR

`get.activated.win` hosts a PowerShell one-liner that downloads and executes the Microsoft Activation Scripts (MAS) all-in-one activation toolkit. The page's content is essentially the launcher script itself: it verifies PowerShell language mode, checks .NET availability, downloads MAS_AIO.cmd from one of three mirrors (GitHub raw, Azure DevOps, or a self-hosted git), SHA256-verifies the payload against a pinned hash, and then runs it with elevated privileges.

## Key claims

- Script downloads from three mirrors (GitHub, Azure DevOps, self-hosted) in random order to improve reliability and avoid single-point-of-failure; fails with a helpful error if all three are unreachable.
- SHA256 integrity check (`D94B1ABCBA24D26C5FBE114A15B53A558684D74A1ACCFF79BBB2407BE7102A89`) is performed before execution — the script aborts if the hash mismatches, linking users to a troubleshooting page.
- Checks for 3rd-party antivirus (`SecurityCenter2` WMI) and AutoRun registry keys that commonly cause CMD crashes, printing targeted remediation instructions before aborting.
- Supports both x64 and x86 PowerShell (PSv2 path uses `saps` with Sysnative detection); requires PowerShell Full Language Mode (not Constrained).
- The script is for [[Microsoft]] Windows activation only; content is entirely a self-contained PowerShell launcher with no ML relevance.
