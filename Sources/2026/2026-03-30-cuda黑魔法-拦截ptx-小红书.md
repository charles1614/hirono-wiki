# [2026-03-30] CUDA黑魔法 ｜拦截PTX

## TL;DR

BobHuang documents a tool (dsl-learn/cuda-magic) that intercepts and captures PTX output from [[CUDA]], [[OpenAI Triton]], [[CUTLASS]], cuTile/CuteDSL kernels, enabling inspection of the lowest compiler-visible layer of GPU optimization.

## Key claims

- PTX (Parallel Thread Execution) is described as "NVIDIA留给用户优化算子的最后一层" — the deepest optimization layer accessible to users above machine code.
- The tool intercepts PTX emitted by CUDA, [[OpenAI Triton]], [[CUTLASS]], cuTile/CuteDSL and stores it for inspection; open-sourced at dsl-learn/cuda-magic.
- Framed within an observation about the evolution of programming tooling: OJ → LLM dialogue → Agent-based delegation, with the author noting that reducing context consumption is now the key developer discipline.
