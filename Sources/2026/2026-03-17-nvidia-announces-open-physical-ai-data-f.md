# [2026-03-17] NVIDIA Announces Open Physical AI Data Factory Blueprint to Accelerate Robotics

## TL;DR

Announced at GTC 2026, the **NVIDIA Physical AI Data Factory Blueprint** is an open reference architecture that unifies data curation, synthetic data generation, and evaluation for physical AI systems (robotics, vision AI agents, autonomous vehicles). It integrates [[NVIDIA Cosmos]] world foundation models, [[NVIDIA OSMO]] orchestration, and cloud-provider partnerships (Microsoft Azure, Nebius) to turn accelerated compute into high-volume, model-ready training datasets. NVIDIA's framing: "compute is data."

## Key claims

- **Blueprint unifies three pipeline stages** into a single automated workflow: (1) *Curate and Search* via Cosmos Curator (open-sourced on GitHub); (2) *Augment and Multiply* via Cosmos Transfer — expands real and simulated data to cover rare long-tail edge cases; (3) *Evaluate and Validate* via Cosmos Evaluator (now on GitHub), powered by Cosmos Reason.
- **Physical AI follows scaling laws** — NVIDIA explicitly applies the model: performance improves as data, compute, and model capacity grow. The factory blueprint is the data-side infrastructure to enable that scaling for robots and AVs.
- **NVIDIA OSMO** (open-source orchestration) now integrates with leading coding agents — Claude Code, OpenAI Codex, and Cursor — enabling AI-native infrastructure management: proactive resource allocation, bottleneck resolution, automated pipeline delivery.
- **Cloud-provider integrations**: Microsoft Azure integrates the blueprint into an open physical AI toolchain (on GitHub) with Azure IoT Operations, Microsoft Fabric, Real-Time Intelligence, and Microsoft Foundry. Nebius integrates OSMO into its AI Cloud, powered by RTX PRO 6000 Blackwell Server Edition GPUs, adding native data management, labeling, serverless execution, and managed inference.
- **NVIDIA Alpamayo** — the world's first open reasoning-based VLA (vision language action model) for long-tail autonomous driving — is NVIDIA's internal use case for the blueprint. [[Skild AI]] uses it for general-purpose robot foundation models; Uber uses it to accelerate AV development.
- **Ecosystem early adopters**: FieldAI, Hexagon Robotics, Linker Vision, Milestone Systems, RoboForce, Teradyne Robotics, and Uber across Azure and Nebius infrastructure. Use cases span industrial perception, mobility, reinforcement learning pipelines, video analytics AI agents, and industrial humanoid robots.
- Blueprint expected on GitHub in April 2026.
