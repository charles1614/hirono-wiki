# [2025-07-06] LLMs-from-Scratch: Build a Large Language Model (Sebastian Raschka)

## TL;DR

Official companion repository for Sebastian Raschka's book "Build A Large Language Model (From Scratch)" (Manning, 2024). Seven chapters take readers from text data loading through GPT implementation, pretraining, and instruction finetuning via DPO, with bonus material covering MoE, MLA, GQA, sliding-window attention, and architectures including Llama 3.2, Qwen3, Gemma 3/4, and Olmo 3.

## Key claims

- Book published Manning 2024, ISBN 978-1633437166; companion video course is 17 hours 15 minutes at Manning LiveVideo.
- Ch 1–7 flow: LLM concepts → text data (BPE tokenizer) → attention mechanisms → GPT from scratch → pretraining on unlabeled data → classification finetuning → instruction finetuning with DPO.
- Bonus appendices cover LoRA (Appendix E), DDP multi-GPU training (Appendix A), and exercise solutions (Appendix C).
- Extensive bonus chapters add: KV cache, MoE, GQA, MLA, sliding-window attention, GatedDeltaNet, and from-scratch implementations of Llama 3.2, Qwen3 Dense+MoE, Gemma 3, Olmo 3, Tiny Aya, Qwen3.5, and Gemma 4.
- A sequel book "Build A Reasoning Model (From Scratch)" covers inference-time scaling, GRPO RLVR, distillation, and LLM-as-judge evaluation starting from a pretrained model.
- All code designed to run on conventional laptops; GPU auto-detected but not required.
