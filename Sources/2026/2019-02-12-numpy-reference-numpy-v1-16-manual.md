# [2019-02-12] NumPy Reference — NumPy v2.4 Manual

## TL;DR

Official [[NumPy]] reference manual index page listing the full Python API, C API, and supplementary topics (CPU/SIMD optimizations, thread safety, global configuration). Serves as the top-level navigational entry point for NumPy's programmatic surface.

## Key claims

- Python API organized into: array objects (ndarray, scalars, dtypes, iterating, masked arrays, datetime), universal functions (ufunc), and routines by topic (FFT, linear algebra, random sampling, statistics, sorting, string, I/O, indexing, polynomials, and more).
- C API covers Python type/C-structure bindings, data type API, array API, ufunc API, generalized ufunc API, NpyString API, memory management, and C API deprecations.
- Additional topics include Array API standard compatibility, CPU/SIMD optimizations, thread safety, global configuration options, NumPy security, and `numpy.distutils` migration advice.
- Manual originates in part from Travis E. Oliphant's "Guide to NumPy" (public domain since August 2008), augmented by contributor documentation.
