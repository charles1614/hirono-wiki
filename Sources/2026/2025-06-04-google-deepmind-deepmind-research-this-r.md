# [2024] google-deepmind/deepmind-research — Reference Implementation Repository

## TL;DR

[[DeepMind]]'s public repository of reference implementations accompanying research publications, covering 60+ projects spanning reinforcement learning, protein folding, graph networks, generative models, and scientific ML from 2019 to the present.

## Key claims

- The repo accompanies peer-reviewed publications (Nature, ICML, NeurIPS, ICLR) with illustrative code and implementations, not production-grade libraries — flagged as "not an official Google product."
- Notable projects include AlphaFold CASP13 (Nature 2020), MeshGraphNets for physical simulation (ICLR 2021), Perceiver IO for structured I/O (2021), and BYOL self-supervised learning (NeurIPS 2020).
- Covers scientific domains beyond LLMs: tokamak plasma control (Nature 2022), density functional approximation (Science 2021), precipitation nowcasting (Nature 2021), and galaxy merger detection.
- Earlier entries (2019–2020) include foundational RL algorithms: DQN variants (tandem_dqn), option-keyboard skill composition, VISR fast task inference, and offline RL benchmarks (rl_unplugged).
