# [2025-07-23] Wave Terminal — AI-Integrated Open-Source Terminal

## TL;DR

Wave Terminal is an open-source, cross-platform terminal (macOS/Linux/Windows) with a built-in context-aware AI assistant, durable SSH sessions, a graphical file editor, and a flexible drag-and-drop block layout. Licensed Apache-2.0.

## Key claims

- Supports any LLM via BYOK (OpenAI, Claude, Gemini, Azure, Perplexity, Ollama, LM Studio); no account required.
- "Durable SSH sessions" survive network interruptions and application restarts with automatic reconnection — a key differentiator from conventional terminals.
- `wsh` CLI command system lets users manage workspaces from the terminal and share data between sessions; `wsh file` enables seamless file sync between local and remote SSH hosts.
- Secure credential storage uses native OS backends (Keychain on macOS) so API keys are accessible across SSH sessions without plaintext exposure.
- Minimum requirements: macOS 11+, Windows 10 1809+, or Linux glibc-2.28+ (Debian 10 / Ubuntu 20.04 / RHEL 8).
- Sponsors development via GitHub Sponsors; community on Discord.
