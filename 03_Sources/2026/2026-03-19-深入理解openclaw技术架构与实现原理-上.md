# [2026-03-19] 深入理解OpenClaw技术架构与实现原理（上）

## TL;DR

Aliyun Developer WeChat account deep-dive into OpenClaw's internal architecture (Part 1), authored collaboratively with AI agents (OpenClaw, Claude Code, NotebookLLM, DeRisk). Covers the Gateway control plane, Pi/Agentic Loop inference engine, cron system, tool system, Channels integration, context management, SubAgent architecture, and sandbox system across 16 named modules.

## Key claims

- [[OpenClaw]]'s architecture is "local-first, multi-device" with a WebSocket **Gateway** as the single control plane — default port 18789, supports loopback/LAN/Tailscale/custom bind modes. — Source §3.1
- The Gateway's authentication modes are `token`, `password`, `trusted-proxy`, and `device-token`; plaintext `ws://` to non-loopback addresses is blocked (CWE-319 enforcement). — Source §3.1.5
- The **Pi Agent / Agentic Loop** is event-driven: outer `run.ts` handles retries and profile rotation; inner `attempt.ts` manages one full LLM-call lifecycle; `subscribe.ts` dispatches streaming events; the tool loop is fully SDK-managed (auto-executes `tool_use` and continues calling). — Source §3.2
- LLM backend is pluggable via `streamFn`; default is `streamSimple` from `@mariozechner/pi-ai`, with an Ollama override path (`createOllamaStreamFn`). — Source §3.2.3
- The **CronService** (`src/cron/service.ts`) supports three schedule types: `at` (one-shot), `every` (fixed interval), and `cron` (standard cron expression with timezone + stagger); max timer delay is capped at 60 s. — Source §3.3
- **Channels** natively integrate WhatsApp, Telegram, Slack, Discord, Google Chat, Signal, iMessage, Microsoft Teams, Matrix, and Zalo; group routing includes mention gating, reply-tag handling, and per-channel auto message splitting. — Source §2
- **SubAgent sandbox**: non-main sessions (group or external channel) can be isolated in separate Docker containers with tool allowlist/blocklist; DM pairing requires a code by default to prevent untrusted input. — Source §2.6
- The article is self-described as having been written in collaboration with OpenClaw, Claude Code, and other agents — an example of AI-assisted technical writing about the AI tool itself. — Source §1
