# [2019-02-12] Python Module Index — Python 3 Standard Library

## TL;DR

Official Python 3 standard library module index, providing an alphabetical listing of all stdlib modules with one-line descriptions. Useful as a navigational reference for the full scope of built-in Python modules across versions 3.7 through 3.14.

## Key claims

- Covers the complete standard library from `__future__` and `__main__` through `zipimport`, with per-module links and deprecation/removal notices (e.g., `aifc`, `asynchat`, `asyncore`, `cgi`, `chunk`, `audioop` all marked "Removed in 3.12/3.13").
- Key modules include `asyncio` (asynchronous I/O), `collections` (container datatypes), `concurrent.futures`, `dataclasses`, `importlib`, `multiprocessing`, `pathlib`, `sqlite3`, `typing`, `unittest`, and `urllib`.
- Module index is alphabetically indexed (_, a–z sections) as an aid for fast navigation.
