# [2026-03-16] 我们撞车了Kimi的注意力残差，但用在压缩上

## TL;DR
An author describes how their context-compression paper "Context Compression via Explicit Information Transmission" independently converged on the same architectural intuition as [[Kimi K2]]'s Attention Residuals: both diagnose progressive representation dilution across layers and both propose residual attention over hidden states as the fix, but applied to compression rather than base model design.

## Key claims
- Both Kimi's Attention Residuals and the author's compression work diagnose "progressive dilution" / "progressive representation overwriting" — deep layers overwrite features from earlier layers that are worth preserving.
- Their compression approach uses residual attention to extract [[Attention Residual]]-style cross-layer hidden states directly from a frozen LLM's forward pass, bypassing LLM fine-tuning and the dilution path.
- Residual attention tends to select features from early-to-middle layers, confirming that deep-layer representations are unsuitable for compression.
- Optimal Transport is added to resolve redundancy in compressed attention allocation.
- The paradigm treats the LLM as a feature extractor, opening broader design space without modifying self-attention internals.
