# [2026-03-15] 终端里直接看图Ghostty必装文件管理器Yazi

## TL;DR
A practitioner tip recommending Yazi — a terminal file manager — as an essential companion for the Ghostty terminal, highlighting its inline preview capability (code with syntax highlighting, Markdown, PDF, images, video) and full keyboard-driven navigation.

## Key claims
- Yazi renders inline previews of code (syntax-highlighted), Markdown, PDF, images, and video directly in the terminal split pane without opening an editor.
- Full keyboard navigation: hjkl for directory traversal, standard vim-style bindings for file operations (a/r/y/p/x/d).
- Integrates with fzf (fuzzy file search via `z`) and zoxide (frecency jump via `Z`); exits to the last-browsed directory automatically.
- Author reports replacing Finder entirely after adopting Yazi with Ghostty.
