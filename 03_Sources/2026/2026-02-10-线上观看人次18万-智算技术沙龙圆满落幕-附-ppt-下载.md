# [2026-02-09] 线上观看人次18万+！智算技术沙龙圆满落幕（附 PPT 下载）

## TL;DR
2026年1月31日由 SGLang、阿里云、龙蜥社区联合主办的智算技术沙龙会议纪要。汇聚国内顶尖 AI Infra 团队，深度覆盖 KVCache 优化、PD/EPD 分离部署、AgenticRL 训练框架、多模态推理架构等前沿议题，线上观看超18万人次。

## Key claims
- [[SGLang]] 核心开发者蔡尚铭介绍2025年技术演进里程碑：PD 分离大规模部署、KVCache 分层缓存、RL 集成、Encoder-Prefill-Decode 分离、面向超长上下文的分块流水线并行加速。
- [[Mooncake]] 通过零拷贝传输、多网卡池化与弹性扩展，助力 SGLang 实现 PD/EPD 分离与分布式 KVCache 共享；KTransformers 基于 AVX/AMX 指令集实现 CPU/GPU 异构推理，支持 NVFP4、FP8、BF16 原生精度 MoE kernel。
- 阿里云 ROLL 强化学习框架通过异构硬件亲和性调度、细粒度异步编排、状态感知弹性部署，已在3000卡集群、千亿参数 MoE 模型上实现生产级吞吐。
- Slime RL 训练框架（清华大学谢承兴）针对 AgenticRL 场景优化：灵活 rollout 机制、解耦 agent 接入、高效并行与同步策略，提升 AgenticRL 可扩展性。
- EPD（Encoder-Prefill-Decode）三段分离在图像密集型请求下（单次4–8张图），1 QPS 负载下延迟降低约6–8倍，高 QPS 下吞吐提升约2倍。
- 阿里云 TairKVCacheManager 是已开源的企业级全局 KVCache 管理系统，原生支持 Mooncake，面向 Agent 时代多轮对话的 KVCache 存储需求演进。
- 圆桌讨论涵盖：国产 GPU 在量化与存储访问的创新潜力、CXL/RDMA 在跨节点 KV 传输应用、稀疏 Attention 工业落地、超节点分层存储体系演进。
