# [2026-02-14] Qwen3.5: Towards Native Multimodal Agents

## TL;DR
Alibaba announces Qwen3.5-397B-A17B, an open-weight native vision-language MoE model that is the first in the Qwen3.5 series. Built on a hybrid Gated DeltaNet + Gated Attention architecture with 397B total / 17B active parameters, it matches the >1T-parameter Qwen3-Max base while delivering 8.6×–19× higher decoding throughput. The model is natively multimodal, covers 201 languages, and targets universal agentic use cases including GUI automation and coding.

## Key claims
- Architecture is a hybrid sparse MoE fusing linear attention (Gated Delta Networks) with standard Gated Attention, with 397B total parameters and only 17B activated per forward pass — inference throughput is 8.6× (32k context) to 19× (256k context) that of Qwen3-Max. — [[Qwen]]
- Qwen3.5-397B-A17B matches Qwen3-Max-Base (>1T parameters) despite activating 17B per pass; this is attributed to larger-scale visual-text pretraining, stricter filtering, and the Gated DeltaNet hybrid. — [[Qwen]]
- A 250k-token vocabulary (up from 150k) improves encoding/decoding efficiency by 10–60% across most languages; multilingual support grows from 119 to 201 languages/dialects. — [[Qwen]]
- A fully disaggregated asynchronous RL framework achieves 3×–5× end-to-end RL speedup, supporting million-scale agent scaffolds through FP8 end-to-end training, rollout router replay, speculative decoding, and multi-turn rollout locking. — [[MoE Training]]
- Native FP8 pipeline applies low precision to activations, MoE routing, and GEMM operations with runtime monitoring preserving BF16 in sensitive layers — ~50% activation memory reduction and >10% speedup while scaling to tens of trillions of tokens. — [[FP8 Computation]]
- On agent benchmarks, Qwen3.5-397B-A17B scores: BFCL-V4 72.9, TAU2-Bench 86.7, VITA-Bench 49.7, OSWorld-Verified 62.2, AndroidWorld 66.8. Competitive with GPT5.2 and Gemini-3 Pro across most categories. — raw source
- Qwen3.5-Plus (hosted, 1M context window) integrates natively with [[OpenClaw]] and other coding agents for vibe coding. — [[OpenClaw]]
