# [2026-03-24] Z.AI Model Release Notes

## TL;DR

Z.AI (the commercial brand of Zhipu AI) publishes a chronological release-notes changelog for its GLM model family, spanning mid-2025 through April 2026. The series shows a clear trajectory from coding/reasoning foundation models toward long-horizon agentic models culminating in GLM-5.1, which targets 8-hour autonomous engineering tasks.

## Key claims

- [[GLM-5.1]] (2026-04-07): designed for long-horizon tasks, can work independently up to 8 hours in a single run; claims comprehensive capability alignment with Claude Opus 4.6; built with multi-turn SFT, RL, and process-quality evaluation.
- GLM-5V-Turbo (2026-04-01): vision-based coding and GUI agent workflows; native multimodal (images, video, text); targets planning, coding, and action execution in visually grounded tasks.
- GLM-5-Turbo (2026-03-15): high-throughput long-chain agent stability; improves tool/Skills integration and complex instruction decomposition.
- [[GLM 5]] (2026-02-12): complex system engineering and long-range agent tasks; benchmarks against Claude Opus 4.5 in code-logic density; integrates DeepSeek Sparse Attention for higher token efficiency.
- GLM-4.7 (2025-12-22): open-source SOTA on coding/reasoning benchmarks at launch; enhanced agentic coding for multi-step tasks.
- GLM-4.5 (2025-07-28): native agentic LLM with doubled parameter efficiency; one-click compatibility with Claude Code framework.
- AutoGLM-Phone-Multilingual (2025-12-11): multimodal mobile automation via ADB; natural-language task execution across 50+ apps; supports English and Chinese.
